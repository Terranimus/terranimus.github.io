What's shipped

TerranSoul doesn't publish a formal numbered-release changelog yet — development happens directly on main through dated, checkpoint-tagged commits. This page distills that history rather than inventing a version scheme that doesn't exist. Dates are commit dates on TerranSoulApp's main branch.

This list only includes work confirmed merged to main at the time this page was written. It is not exhaustive of every commit — see the full commit history for that.

2026-07-02

CP-1

Headless / always-on resident host mode. The full brain/memory stack runs with no Tauri window, exposed over the existing MCP JSON-RPC-over-HTTP transport — a dedicated terransoul-headless binary, plus --headless on the main binary, with Windows service install/status/uninstall.

CP-2 / CP-3

Bidirectional chat channels. Telegram (poll-based), then Discord (persistent Gateway WebSocket), Slack (Socket Mode), and WhatsApp (Baileys sidecar) — one shared adapter contract, four platform-specific auth flows. See Channels.

CP-4

brain_execute_code. A sandboxed code-execution MCP tool with bounded output and procedural-ledger learning, gated by the earned-autonomy action-trust ladder. See Agentic coding.

CP-5

brain_pursue_goal. A persistent, standing-objective pursuit loop built as orchestration over the existing goal stack, confidence gate, and reasoning-budget allocator. See Goals & automation.

CP-6

Learning-timeline UI (read-only). A scrubbable timeline over existing procedural-confidence, skill-library, and knowledge-graph provenance data — no new table, a pure read projection.

CP-7

Agent-drivable Canvas panel. A sandboxed secondary window (ts-canvas:// custom scheme, session-root jailed) an agent can navigate, evaluate JavaScript in, and snapshot — canvas_navigate / canvas_eval / canvas_snapshot.

Marketplace

External-agent plugin catalog. A 223-entry catalog generated from the Hermes Agent and OpenClaw plugin trees, with a consent-first install/uninstall flow dispatched through the agentic-coding roster. See Marketplace.

BCF-21

Write-time governance wiring. The governed/safe self-evolving-memory gate (drift, poisoning, and growth guardrails) wired into the real connector and peer write paths, plus a persistent-injection regression suite.

BRAIN-SEARCH-1

brain_search rerank defaults to off. A measured production defect — the LLM-as-judge rerank stage added 4.3–18.9s per call and could score every in-corpus candidate below threshold — was fixed by defaulting rerank off (raw RRF/hybrid recall alone, 355–2325ms) and making the opt-in rerank fail-open. brain_delete_memory (scoped single-id delete) shipped alongside it.

2026-06-30

Wave 5

Cognitive-faculties roster complete (28 chunks, schema V58). BCF-06, BCF-07, BCF-16, BCF-17, BCF-21, plus CA-3 (automation blueprints) and CA-6 (proactive commitments).

2026-06-29

Wave 4

21 cognitive faculties + multi-hop retrieval (schema V51). Includes BCF-21 governed/safe self-evolving memory — drift, poisoning, and unbounded-growth guardrails on what the brain writes to itself.

2026-06-28

BCF cognitive faculties

First cognitive-faculty chunks: metacognition (confidence calibration), drives, the goal stack, and bias-audit.
