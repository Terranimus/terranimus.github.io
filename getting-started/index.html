Getting started

This is the fast path: open the app, say something, see where things live. For the non-technical explanation of why any of this exists, read TerranSoul in Six Scenes instead — this page assumes you already installed it and just want to get moving.

1. Open the app and pick a brain

First launch drops you into a short wizard that asks how to power the brain — free cloud API, paid cloud API, or a local model via Ollama/LM Studio/llama.cpp/an existing Claude CLI subscription. If you haven't installed yet, see Installing TerranSoul. Pick whichever fits your machine and budget; you can change it later from the Brain tab.

2. Have a first conversation

The app opens on the Chat tab. Type something and send it — that's it. There's nothing to configure before your first message goes through; retrieval, memory writes, and the rest of the pipeline run automatically underneath.

A few things worth noticing on that first exchange:

  • Replies stream in as they're generated, not all at once.
  • Anything worth remembering from the conversation gets written to memory in the background — you don't have to ask it to "remember" something explicitly, though you can.
  • If you close the app and come back tomorrow, the context from today is still there.

3. Find the other tabs

The sidebar has more than chat. The ones worth knowing about early:

Brain

An interactive map of the brain-region modules — focus/triage, hybrid recall, memory tiers, affect, skills — all wired to one MCP nerve. Also where you switch/reconfigure the model.

Knowledge

The memory surface itself: search what's been remembered, inspect the knowledge graph, switch the Memory Graph view to 3D. This is where brain_search/brain_delete_memory live for a human, not just an agent.

Market

The plugin marketplace tab — browse and install the external-agent plugin catalog.

Channels (backend)

Not yet a dedicated settings tab — configured today via the channel_save_credential/channel_list commands. Wire up Telegram, Discord, Slack, or WhatsApp so TerranSoul can talk to you there too.

4. Where your data lives

Everything — chat history, memories, the knowledge graph, credentials (encrypted) — is stored locally under the app's data directory (bundle id com.terransoul). Nothing is uploaded anywhere unless you explicitly configure a cloud model provider, in which case only the prompt/response for that turn goes to that provider, same as any other AI app using a cloud API.
